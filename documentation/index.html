Installation

Just download the project to anywhere in your computer and then.

  1. Open script folder in a Code Editor.
  2. Install All dependencies : 
    composer install
  3. Create a new database, for example with phpMyAdmin. Then open the .env.example file, edit it to match your database name, username and password and save it as .env file. Once done, build tables with the following command: 
    php artisan migrate
  4. Now fill the tables: 
    php artisan db:seed
  5. Finally, generate the application key 
    php artisan key:generate
  6. Run the artisan serve command : 
    php artisan serve
  7. Open your favorite browser and Proceed to : http://localhost:8000/students
